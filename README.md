# Codeforces Exercises

Repositório com minhas soluções de exercícios do [Codeforces](https://codeforces.com/), desenvolvidas em **C** para treino de lógica, estruturas básicas e resolução de problemas competitivos.

## Estrutura

Cada problema fica em uma pasta própria:

```text
├── Codeforces 1154A/
│   └── codeforces1154A.c
├── Codeforces 151A/
│   └── codeforces151A.c
...
```

```bash

```
